cli-socnet
==========

Exercise - implement a console-based social networking application with basic post / read / follow / wall functionality

Pre-reqs
--------
* Java 7
* lein

Running
-------
### Option 1
```
lein run
```

### Option 2
```
lein uberjar
java -jar target/cli-socnet-0.1.0-SNAPSHOT-standalone.jar
```
